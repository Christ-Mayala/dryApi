# 📥 Guide de téléchargement des diagrammes

## Méthode SUPER SIMPLE (recommandée !)

### Étape 1 : Aller sur mermaid.live
Ouvrez votre navigateur et allez sur : **https://mermaid.live/**

### Étape 2 : Copier-coller un diagramme
1. Ouvrez le fichier `.mmd` correspondant au diagramme que vous voulez (ex: `01_architecture.mmd`)
2. Sélectionnez tout le contenu et copiez-le (`Ctrl+A` puis `Ctrl+C`)
3. Allez sur mermaid.live et collez le code dans l'éditeur de gauche (`Ctrl+V`)

### Étape 3 : Télécharger !
1. Cliquez sur le bouton **"Actions"** en haut à droite
2. Choisissez :
   - **"Download PNG"** pour une image (meilleure qualité)
   - **"Download SVG"** pour un format vectoriel (peut être redimensionné sans perte)
